# 🚀 Clipboard Translator: 機能詳細 & 活用ガイド

**「コピーするだけ」で、世界最先端のAIがあなたの隣に。**
このドキュメントでは、Clipboard Translator が具体的に何を可能にするのか、その全貌を詳細に解説します。

---

## 💎 コア・コンセプト: "Zero Friction" (摩擦ゼロ)

既存の翻訳ツールは、「アプリを開く」「貼り付ける」という手間がありました。
このプロジェクトは、**OS標準のコピー操作 (`Ctrl+C`) を拡張する** ことで、翻訳という行為を完全に**無意識化**します。

翻訳ツールを「使う」のではなく、クリップボードが「進化する」感覚です。

---

## 🌟 3つの主要機能 (Deep Dive)

### 1. ⚡ Instant Trigger (瞬間起動)
- **操作**: テキストを選択して `Ctrl+C` を2回連打（500ms以内）。
- **挙動**: マウスカーソルのすぐ隣に、フローティングウィンドウが即座に出現します。
- **メリット**: 視線移動が最小限で済みます。ブラウザのタブを切り替える必要は一切ありません。

### 2. 🧠 Context-Aware Translation (文脈理解AI)
**Google Gemini Pro** モデルを搭載しているため、従来の「単語の置き換え」とは次元が異なります。
- **スラング・専門用語**: 辞書に載っていないネットスラングや、プログラミング特有の用語も文脈から正確に解釈します。
- **文化的背景**: 単に訳すだけでなく、「なぜそういう言い回しをするのか」という背景知識まで解説してくれます。
- **要約機能**: 長文をコピーした場合、翻訳だけでなく「3行要約」も生成可能です。

### 3. 🛡️ Intelligent Mode Switching (スマートモード切替)
状況に合わせて、AIの出力をコントロールできます。

| モード | 特徴 | おすすめの場面 |
| :--- | :--- | :--- |
| **Simple Mode (簡潔)** | 結果を3～5行でズバリ表示。 | チャットの返信、エラーログの確認、急いでいる時。 |
| **Detail Mode (詳細)** | 翻訳文、元の意味、文法解説、類語などを網羅。 | 英語の学習、技術ドキュメントの精読、洋書を読む時。 |

## 🎯 ユースケース別活用法

### 👩‍💻 開発者 / エンジニア
- **エラーメッセージの解読**: 難解な英語のエラーログをコピー。原因と解決策のヒントまでAIが推測してくれる場合があります。
- **ドキュメント参照**: 公式ドキュメントの微妙なニュアンス（SHOULD vs MUSTなど）を正確に把握。
- **変数名の検討**: 日本語で変数名の案をコピーして翻訳させれば、自然な英語の変数名を提案させることも可能（プロンプト次第）。

### 📚 学習者 / 学生
- **多読の補助**: いちいち辞書を引くと集中が切れますが、これなら0.5秒で意味を確認してすぐに本文に戻れます。
- **解説で学ぶ**: 単なる訳ではなく「解説」が表示されるため、読むだけで英語力が向上します。

### 🌏 海外情報の収集
- **海外ニュース / Reddit**: スラングや現地のジョークも、Geminiが補足説明してくれるので深く理解できます。

## ⚙️ 高度なカスタマイズ

`config.json` や設定画面を通じて、自分好みにチューニング可能です。

- **Prompt Engineering**: AIに送るプロンプト（命令文）は内部で最適化されていますが、将来的なアップデートでユーザーごとのカスタムプロンプトにも対応可能です。
- **Model Selection**: `gemini-pro` だけでなく、より高速な `gemini-2.0-flash` など、最新モデルに切り替えてレスポンス速度を極限まで高めることができます。

## 🛠 技術的な優位性 (Why Rust?)

- **超軽量**: Electron製アプリと異なり、メモリ使用量はごくわずか（数MB〜）。常駐させてもPCが重くなりません。
- **堅牢性**: Rustのメモリ安全性により、長時間起動しっぱなしでもクラッシュしにくい設計です。
- **クロスプラットフォーム基盤**: 現在はWindows特化ですが、内部アーキテクチャはLinux/macOSへの移植を見据えて設計されています。
