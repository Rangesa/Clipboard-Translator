# 🔥 Clipboard Translator: 3秒でわかる機能まとめ

**「Ctrl+C」を2回押すだけ。AIがその場で翻訳・解説します。**

## ✨ 何ができるの？

| 機能 | 説明 |
| :--- | :--- |
| **🚀 即座に翻訳** | テキスト選択して `Ctrl+C` `C` (連打) で、マウスの横に翻訳画面がポップアップ。 |
| **🧠 AIが解説** | 単なる翻訳ではなく、**Google Gemini** がスラングや文化的背景のアドバイスもしてくれます。 |
| **⚡ 爆速起動** | Rust製で超軽量。メモリを食わず、Windowsの邪魔をしません。 |
| **🔄 双方向対応** | 英語は日本語へ、日本語は英語へ。自動判定でスマートに変換。 |

## ⚖️ 今までの翻訳 vs Clipboard Translator

- **今まで**: コピー ➔ ブラウザ開く ➔ 翻訳サイト開く ➔ ペースト ➔ 読む 😩
- **本アプリ**: **コピー連打 ➔ 読む** 🤩

## 🛠 2つの表示モード

1.  **詳細モード**: 翻訳・解説・要約をすべて表示。じっくり読みたい時に。
2.  **簡潔モード**: 3行でズバリ要点だけ。急いでいる時に。
